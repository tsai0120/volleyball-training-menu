# 快速開始指南

## 🚀 第一次使用

### 1. 安裝依賴
```bash
npm install
```

### 2. 啟動開發伺服器
```bash
npm run dev
```

### 3. 打開瀏覽器
訪問 [http://localhost:3000](http://localhost:3000)

---

## 📱 使用說明

### 一般模式（預設）

1. **查看訓練時間場地**
   - 點擊「訓練時間場地」分頁
   - 點擊日期卡片查看當天詳情
   - 可看到出席狀況和訓練項目預覽

2. **查看寒訓菜單**
   - 點擊「寒訓三天菜單」分頁
   - 選擇要查看的日期
   - 點擊項目卡片查看詳細資訊
   - 包含：時間、分組、要求、圖示等

3. **查看出席人員**
   - 點擊「寒訓三天出席人員」分頁
   - 按系級分組顯示
   - 可點擊日期查看當天出席狀況
   - 底部有統計資訊

### 管理員模式

1. **切換管理員模式**
   - 點擊右上角「一般模式」按鈕
   - 切換為「管理員模式」

2. **編輯項目**
   - 進入「寒訓三天菜單」分頁
   - 點擊項目卡片的編輯按鈕（✏️）
   - 修改內容後點擊「儲存」

3. **拖移項目**
   - 在管理員模式下，直接拖動項目卡片
   - 可重新排序項目

4. **新增項目**
   - 點擊「新增項目」按鈕
   - 或點擊「休息/吃飯/場佈/場復」快速新增

5. **自動分組**
   - 編輯項目時，設定「分組（幾人一組）」
   - 點擊「自動分組」按鈕
   - 系統會自動將當天出席人員分組

6. **上傳圖片**
   - 編輯項目時，點擊「上傳圖片」
   - 選擇圖片檔案
   - 圖片會顯示在項目詳情中

## 💡 小貼士

1. **圖片建議**
   - 建議使用 PNG 或 JPG 格式
   - 圖片大小建議不超過 2MB
   - 目前使用 base64 編碼，大圖片可能影響載入速度

2. **分組功能**
   - 自動分組會隨機分配人員
   - 可以手動調整分組名單
   - 分組資訊會儲存在項目中

3. **數據儲存**
   - 目前數據儲存在前端
   - 重新整理頁面會恢復原始數據
   - 未來可整合資料庫實現持久化

## 🆘 遇到問題？

1. **無法啟動？**
   - 確認已執行 `npm install`
   - 確認 Node.js 版本 >= 18

2. **樣式顯示異常？**
   - 確認已安裝 Tailwind CSS
   - 重新執行 `npm install`

3. **拖放功能不工作？**
   - 確認已安裝 react-dnd
   - 檢查瀏覽器是否支援 HTML5
